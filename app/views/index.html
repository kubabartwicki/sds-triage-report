Export triage report

This tool takes data from Zendesk and exports it to HTML. The purpose of this is to create a triage report from a Service Design Support case, so we can show to our stakeholders how our support looked like.

To use, go to /ticket/{ID}, swapping {ID} with the ID of your ticket.

It currently uses the GOV.UK Sandbox. You can edit this and add a token in routes.js.
